How this page works (for site maintainers)

Drop CSV/PNG files in /polls_and_results/data/ (or /data/) and list them in a manifest. The page fetches and renders automatically.

  1. Upload CSVs like /polls_and_results/data/preseason-coaches.csv.
  2. Upload images like /polls_and_results/data/INV1-results.png.
  3. Create polls.json and register each dataset.

CSV columns are flexible. The first row must be headers.
